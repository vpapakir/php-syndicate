1. Installation

Step 1: We create an empty database, preferably under the name "phpsyndicate".

Step 2: We create a database user, and we provide full privileges over the Database:

Step 3: We run the setup script, which is under http://<host>/<path-to-source>/setup/

We press “Continue”.

Step 4: We wait for the installation script to perform checks for the presence of necessary server modules.

Upon success, we press “Continue”.

Step 5: We enter the database credentials. Before we press “Continue” we can check the validity of the credentials by pressing “Verify Settings”.

Step 6: Then, the database is populated with the necessary tables. Upon success, we press “Continue”.

Step 7: Then the database is populated with some sample data. Upon success, we press “Continue”.

Step 8: We can the enter some useful information about the phpSyndicate instance we are installing. We press “Continue”.

Step 9: The configuration file is then generated. Upon success, we press “Continue”.

Step 10: We create the administrative account and press “Continue”.

Step 11: Then, useless installation data are removed. Upon success, we press “Continue”.

The installation process is now completed. By pressing “OK”, user is redirected to the login page.
