The application is best viewed with Mozilla Firefox. With Internet Explorer, some minor elements might look misplaced. It was developed with PHP under WAMP, but it can be used on Linux servers as well, given they are properly configured.

The database schema, along with some sample data, is given in phpmassmailer.sql. The database must be created before the application is used. Should I make a setup process, so we can automate the process of installing?

The first thing user sees, is the index.html where is prompted for username and password. I used the “admin”-“admin” credentials, which are also stored in the database. Should I create a registration process?
